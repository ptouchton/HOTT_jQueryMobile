Responsive Panel (custom breakpoint)

This panel uses a hottPanelBreakpoint class on its page wrapper, along with a custom media query, to create a responsive panel.  In viewports below 45 ems wide, the panel behaves as normal.  In viewports 45 ems and wider, the page reflows to display all of its content, so that the user can interact with the panel and page at the same time.  In addition, the "tap outside to close" functionality is disabled.
