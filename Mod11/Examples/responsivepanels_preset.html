Responsive Panel (preset breakpoint)

This panel uses a ui-responsive-panel class on its page wrapper to create a responsive panel with a preset breakpoint.  In viewports below 55 ems wide, the panel behaves as normal.  In viewports 55 ems and wider, the page reflows to display all of its content, so that the user can interact with the panel and page at the same time.  In addition, the "tap outside to close" functionality is disabled (for the "push" display mode only).
